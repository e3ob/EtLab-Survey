# EtLab-Survey
Complete pending surveys of etlab.
Made for ICET-etlab, might work for others too.
## Usage
### Clone Repository
```sh
git clone https://github.com/e3ob/EtLab-Survey.git
cd EtLab-Survey
```
### Fill envs
```sh
cp sample.env .env
```
and edit the .env file.
**Quotes are are compulsory !!**
